🐢 turtleってなんだ？

turtle (タートル) は、画面上を動き回って 「線を描くことができるカメさん🐢」のことだよ！

Colabでプログラムをコピーしよう

💻 Google Colaboratory（コラボ） を使ってみよう！


Python とは
コンピュータ🖥を操作するための魔法の言葉

コラボで簡単にプログラミングができるよ！

🚀 プログラミングを始めよう！

1. 最初の呪文（じゅもん）を唱えよう！


import turtle

これは、turtle を使うための 【おまじない】だよ！

🐢 カメさんを動かしてみよう！

2. forward() で前に進もう！


forward(100)

これで、カメさんが 「100歩 前に進む」 よ！

🔄 みぎに曲がってみよう！

3. right() でみぎに曲がろう！


right(90)

これで、カメさんが 「90度 みぎに曲がる」 よ！

⬜ 正方形を描いてみよう！

4. forward() と right() を組み合わせて、正方形を描こう！


forward(100)
right(90)
forward(100)
right(90)
forward(100)
right(90)
forward(100)
right(90)

🎨 色を変えてみよう！

5. pencolor() で色を変えよう！


pencolor((255, 0, 0))  # 赤🟥

カッコの中身は (赤🟥, 緑🟩, 青🟦) の順番で、それぞれ 0～255 の間の数字だよ！

🌈 いろんな色を作ってみよう！

  • (0, 0, 255): 🟦青
  • (0, 255, 0): 🟩緑
  • (255, 255, 0): 🟨黄色
  • (255, 128, 0): 🟧オレンジ

好きな色を組み合わせて描いてみよう！

🪄 ペンを上げてみよう！

6. penup() で線を引かずに移動しよう！


penup()
forward(50)

これで、カメさんが 線を引かずに 50歩 前に進む よ！

🖋 ペンをおろそう！

7. pendown() で再び線を引こう！


pendown()

これで、カメさんが 再び線を描く ようになるよ！

💫 ほしがたをかいてみよう！


for i in range(5):
  forward(100)
  right(144)

forをつかうと、おなじ動作をくりかえす ことができるよ！
